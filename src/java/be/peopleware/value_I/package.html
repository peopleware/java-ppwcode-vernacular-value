This package contains guidelines for implementing value type classes. General things are defined in class Value, MutableValue, ImmutableValue and EnumerationValue.

This package contains all value type classes with general applicabiltity defined in different projects. For each value type class, there is an accompanying PropertyEditor. These are placed next to the value type classes they serve, because that is the default location where JavaBeans aware code looks for them.

All value type classes also have accompanying UserType meta-data classes for use of the types with Hibernate. These classes are gathered in subpackage hibernate.
