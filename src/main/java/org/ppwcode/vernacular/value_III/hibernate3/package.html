To persist objects emulating values using Hibernate, they have to be mapped to columns in a {@link java.sql.PreparedStatement SQL update query} and from columns in a {@link java.sql.ResultSet SQL result set}. In Hibernate 3, this is done through {@link net.sf.hibernate.UserType UserTypes}, which have to be registered in the Hibernate config files. This package adds support to make the implementation of such {@link net.sf.hibernate.UserType UserTypes} easy for value types that follow our value vernacular, at least up to a point.
